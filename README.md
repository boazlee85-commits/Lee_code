# Game Code Editor

A modern, feature-rich code editor built with React and Vite for game development and coding projects.

## Features

- 📝 Code editor with syntax highlighting
- 🎮 Game-focused development environment
- 🤖 AI-powered code assistance (integrable)
- 📁 File explorer and project management
- 🎨 Beautiful UI with Tailwind CSS
- ⚡ Fast development with Vite
- 🔐 Authentication support
- 📊 Charts and data visualization

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/game-code-editor.git
   cd game-code-editor
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start the development server
   ```bash
   npm run dev
   ```

The app will be available at `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
- `npm run lint:fix` - Fix ESLint issues
- `npm run typecheck` - Check types with TypeScript
- `npm run preview` - Preview production build
